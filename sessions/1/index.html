To get all these tools under our belts, take our first stab at using HTML, CSS, and Javascript, and to take a go at telling our first story, we are going to populate the people page of the course website with information about ourselves.

  1. Set up all the crap 
  2. Clone the course site repository 
  3. Try running jekyll serve  in the repository at the command line
  4. And if it works, go to http://localhost:4000/ 
  5. Then modify people/index.html and assets/css/people.css to style your own card
