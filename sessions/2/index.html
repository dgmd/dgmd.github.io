What is git/GitHub
and how will we use it?

Interacting with our journals using git/GitHub

  1. Download the GitHub Desktop Client
  2. Open the client and login using your GitHub username and password
  3. Click through File > Clone Resository and then type in the name of your journal's repository (probably USERNAME.github.io)
  4. Click Clone Repository
  5. Now choose where you want the repository to be cloned and what you want it to be called, and click Clone again
  6. Now, using Finder, locate the new repository on your computer and open your journal's index.html file in both Sublime Text and Google Chrome
  7. Make some changes and push them to GitHub

The Pomes Project

Cloning the pomes project with git/GitHub

  1. Go to github.com/dgmd/a-story-in-pomes and fork the repository
  2. Open Github Desktop
  3. Click through File > Clone Resository and then type in the name of the pomes repository (a-story-in-pomes)
  4. Click Clone Repository
  5. Now choose where you want the repository to be cloned and what you want it to be called, and click Clone again
  6. Now, using Finder, locate the new repository on your computer

Viewing the pomes project locally to edit

  1. Open Terminal; change directory into a-story-in-pomes directory.
  2. Run the command git checkout gh-pages; you should now see files in the pomes repository.
  3. Run the command npm install; this installs the various packages required for a-story-in-pomes
  4. Run the command node server.js; this will serve up a version of the pomes site locally.
  5. Now, in Google Chrome, go to the url localhost:3000; this page will reflect any changes you make to the pomes project files on your computer.
  6. Now, try making a change to the index.html file in the pomes repository and saving it to make sure the changes are being reflected.
