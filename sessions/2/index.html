Sharing your 2+2's

If you're comfortable, we recommend creating a new GitHub repository named $USERNAME.github.io—where $USERNAME is your GitHub username—and placing a Markdown file in it named sketchbook.md, where you can record your project ideas and inspirations.

That said, we mostly want you to share your ideas with us in a form and format easy and comfortable for you. Meaning if you prefer a Google Doc (or email), that's fine by us too. The $USERNAME.github.io repository just makes it easy to later leverage GitHub's Pages infrastructure, if you so desire.

If and when you're interested in sharing your ideas and inspirations with the rest of the class (for discussion, critique, or feedback), please do so on the Disource forum here (for project ideas) and here (for inspirations).

back to the /people

  1. Fork the dgmde15.github.io repository into your own user account by clicking "fork" at the upper right. You can read more about forking here or watch this video
  2. Clone the forked repository from your own account
  3. Edit assests/people.css & people/index.html files and add a card with your name
  4. Push your changes to the forked repository with git push origin master at the terminal
  5. Make a pull request to the original site repository. You can read more about making pull requests here or watch this video
