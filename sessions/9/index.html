Exploring L3D via Processing

NB You can take a look at this Instructable for more details.

  1. Download Processing
  2. Install the L3D Processing library by placing the unzipped L3D.zip in ~/Documents/Processing/libraries and restarting Processing.
  3. You should now be able to run any of the included examples (which are also accessible via Processing's File > Examples > Contributed Libraries.
