Object Oriented Gallery

  • Take a look at these examples of Object Oriented effects.

  • Make changes by using the Developer Console functionality in Chrome and by modifying the files HTML/CSS/JavaScript files directly and reloading the page in your browser.

Refining Letterpress

  • If you already cloned our in-progress version of Letterpress last week, pull the changes from the class repository.
  • If you don't have your own copy, you can clone the repository using https://github.com/dgmde15/Letterpress
  • Open up the HTML, CSS, and JavaScript files and poke around.

Extending Letterpress v0.1

If you'd like to keep working with last week's, lighter-weight version, you can checkout the v0.1 tag with git checkout v0.1. To refresh your memory, some possible extensions from last week:

  • Populate the board with a random set of letters.
  • Check if the letters played form an allowable word.
  • Fix it so the color grid doesn't "break" when a letter is played.
  • Create a "reset" button to return the played tiles to their places.

Extending Letterpress v0.2

  • Display the current score in the game
  • Add the ability to re-order and reset individual letters
  • Add the 'surrounded' cell behavior
  • Add the ability to save and restore a game
