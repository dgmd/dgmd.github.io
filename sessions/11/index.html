L3D updates

Tessel.io

  1. Set up the Tessel REPL, a command line interface for interacting with the Tessel.
  2. Try toggling and otherwise controlling the four LEDs on the Tessel board using the REPL.
  3. Display the numbers 0–15 using the LEDs on the Tessel board as a binary counter (you can read more about base systems here)
  4. Blink an LED directly plugged into the Tessel by connecting one leg to GND and another to a GPIO pin which you set HIGH and LOW
  5. Turn the LED on/off by pressing a button
  6. Use an LED as a light sensor and log its reading to the terminal at regular intervals
  7. Blink the LEDs in (3) or (4) proportional to the light reading from (6)
  8. Log out the readings from one of the modules that interest you in the terminal
