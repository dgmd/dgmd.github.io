Bits of Letterpress

  • Populate the board with a random set of twenty-five letters.
  • Check if the letters placed in the tray form an allowable word.
  • Fix the alternating color grid so it doesn't "break" when a letter is played.
  • Create a "reset" button returns played tiles to their places on the board when it is clicked.
