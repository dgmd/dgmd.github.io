This Harvard Extension School course introduces computing as an expressive medium. We prototype, exhibit, and critique interactive, computational stories, art, and games presented as web applications and physical prototypes. We use JavaScript as a unifying language for programmatically controlling and combining audio, video, graphics, text, sensors, and actuators in these projects. Libraries like Processing.js, D3.js, and Snap.svg help us create data-rich, interactive visualizations, and Tessel serves as the microcontroller platform for our physical computing work.
